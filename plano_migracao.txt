COMO VOCÊ INSTALOU O POSTGRES NA SUA VPS ORACLE:
--------------------------------------------------------------------------------
Caso precise revisar o que rodou na VPS:
1. Instalou postgres e postgresql-contrib.
2. Criou o DB 'lucaslogistica' e usuário 'admin_lucas'.
3. Liberou conexões externas no postgresql.conf (listen_addresses = '*') e pg_hba.conf.
4. Reiniciou o postgresql.
5. (Lembrete) Liberar porta 5432 no painel web da Oracle Cloud.

--------------------------------------------------------------------------------
PASSOS PARA CONTINUAR QUANDO VOLTAR (NO SEU COMPUTADOR LOCAL):
--------------------------------------------------------------------------------

1. Renomeie o arquivo `.env.example` para `.env` na raiz do seu projeto.
2. Dentro do `.env`, adicione a sua URL do Banco de Dados PostgreSQL configurado na VPS:
   DATABASE_URL="postgresql://admin_lucas:SUA_SENHA_AQUI@IP_DA_VPS:5432/lucaslogistica?schema=public"

3. No terminal da pasta do seu projeto, instale as dependências:
   npm install

4. Envie as tabelas para a sua VPS da Oracle:
   npx prisma db push

5. Execute o projeto para testar:
   npm run dev
